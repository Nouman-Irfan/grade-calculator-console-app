# 📊 Grade Calculator in C++

A simple yet well-structured **Grade Calculator** project developed in **C++** for Programming Fundamentals.
This project reads student data from a file, calculates averages and grades, and generates formatted results in both the console and output file.

---

## ✨ Features

✅ Read student names and marks from file
✅ Calculate student averages
✅ Assign grades automatically
✅ Calculate class average
✅ Save formatted results to `output.txt`
✅ Display complete report on console
✅ Modular programming using functions

---

## 🛠️ Technologies Used

* 💻 C++
* 📂 File Handling (`ifstream`, `ofstream`)
* 🔁 Loops
* 🧠 Conditional Statements
* 📦 Arrays
* ⚙️ Functions

---

## 📥 Input Format

The program reads data from `input.txt`.

Example:

```txt
Ali 78 85 90 88 92
Ahmed 67 70 72 69 75
Sara 95 91 93 97 96
```

---

## 📤 Output

The program generates:

* Student marks table
* Average of each student
* Grade of each student
* Class average

Output is displayed:

* On Console 🖥️
* In `output.txt` 📄

---

## 🏆 Grade Criteria

| Average  | Grade |
| -------- | ----- |
| 90 - 100 | A     |
| 80 - 89  | B     |
| 70 - 79  | C     |
| 60 - 69  | D     |
| Below 60 | F     |

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Nouman-Irfan/GRADE-CALCULATOR.git
```

2. Open project in Visual Studio / Dev C++ / CodeBlocks

3. Make sure `input.txt` exists in project directory

4. Compile and Run ▶️
